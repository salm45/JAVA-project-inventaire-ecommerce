# JAVA-project-inventaire-ecommerce
This is a Java console-based inventory management project.

The application allows the user to:
- Add new products to the inventory
- Update existing product information
- Delete products
- Display the list of products

The project is built using object-oriented programming (OOP) principles.
The code is organized into multiple packages and folders for better structure and readability.

## Technologies
- Java
- Object-Oriented Programming (OOP)

## Purpose
This project was created for learning and practicing Java programming and OOP concepts.
